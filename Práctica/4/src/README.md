## En Windows

**1. Compilar los .java:**

```
.\jade.ps1 compile
```

**2. Correr la GUI:**

```
.\jade.ps1 gui
```

**3. En otra terminal dividida, lanzar el agente:**

```
.\jade.ps1 agente
```

## En Linux

**1. Compilar los .java:**

```
make compile
```

**2. Correr la GUI:**

```
make gui
```

**3. En otra terminal dividida, lanzar el agente:**

```
make agente
```
